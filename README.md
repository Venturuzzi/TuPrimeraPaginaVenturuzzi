# Proyecto Final - Blog MVT con Django

## Autor
Franco Venturuzzi

## Descripción
Este proyecto implementa un blog básico con Django utilizando el patrón MVT que incluye:
- CRUD de páginas (listar, crear, editar, eliminar) mediante la app **blogpages**
- Sistema de usuarios (registro, login, logout, perfil) mediante la app **useraccounts**
- Sistema de mensajería interna entre usuarios (inbox, enviados, redactar) mediante la app **messagesapp**
